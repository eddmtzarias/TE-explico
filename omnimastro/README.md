# OmniMaestro Core

Sistema de IA pedagógica multiplataforma que analiza capturas de pantalla y proporciona explicaciones adaptativas.

## Estructura del Módulo

```
omnimastro/
├── __init__.py
├── core/           # Motor de IA y procesamiento
├── desktop/        # Aplicación Desktop (Tauri)
├── mobile/         # Aplicación Mobile (Flutter)
├── web/            # Aplicación Web (PWA)
└── shared/         # Código compartido entre plataformas
```

## Uso

Ver el roadmap completo del proyecto:
```bash
python scripts/project_guide.py roadmap
```
